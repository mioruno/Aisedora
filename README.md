# TCPView Clone - Professional Network Monitor

## ✅ WinDivert скачивается АВТОМАТИЧЕСКИ!

При сборке CMake автоматически:
1. Скачает WinDivert с GitHub
2. Скопирует .dll и .sys рядом с .exe
3. Всё готово к использованию!

## Требования:
- **Windows 10/11**
- **Права администратора** для запуска (WinDivert требует)

## Запуск:

```powershell
# Правый клик на run.bat -> "Запустить от имени администратора"
```

## Как работает:
- Перехватывает ВСЕ сетевые пакеты через WinDivert
- Считает РЕАЛЬНЫЕ байты для каждого соединения
- Показывает точную статистику с момента запуска приложения

**Ничего не нужно качать вручную - всё автоматически!**
